# Русский стиль перевода для локализации игр (zh-CN → ru-RU)

## 语气与风格 (Tone & Voice)

- **Уровень тона:** Полуформальный, дружелюбный, информативный. Избегать излишней официальности, но сохранять уважительный и профессиональный стиль.
- **Ключевые слова:** Четкость, доступность, мотивация. Текст должен быть понятен широкой аудитории, включая молодых игроков.
- **Рекомендации:** Использовать активный залог, избегать сложных оборотов. Инструкции и описания должны быть лаконичными и побуждать к действию.

## 术语策略 (Terminology)

- **Правила передачи терминов:**
  - **Ключевые игровые термины** (например, "技能", "忍者", "攻击", "属性") всегда переводятся по глоссарию, без вольных интерпретаций.
  - **Имена собственные** (имена персонажей, уникальные названия) — транслитерируются, если нет устоявшегося перевода.
  - **Уникальные игровые механики** — преимущественно смысловой перевод, чтобы сохранить понятность для русскоязычных игроков.
- **Примеры:**
  - "忍者" → "нинзя"
  - "技能" → "навык"
  - "攻击" → "атака"
  - "升星" → "повышение звезды" или "звёздное повышение" (по контексту)
- **Избегать:** Не использовать кальки с китайского, если они неестественны для русского языка.

## 语法规范 (Grammar)

- **Обращение:** Использовать форму "вы" (Вы/ваш) для обращения к игроку, чтобы сохранить уважительный тон.
- **Гендер:** По умолчанию использовать мужской род для описания персонажей и игроков, если пол не указан явно. В случае обращения к игроку — нейтральные формулировки.
- **Число:** Предпочтительно использовать единственное число при обращении к игроку, множественное — для описания групповых действий.
- **Глаголы:** Использовать повелительное наклонение для инструкций ("Выберите", "Используйте", "Получите").

## UI 简洁性 (UI Brevity)

- **Ограничения по символам:** Стремиться к краткости, не превышать длину оригинального текста более чем на 20%. При необходимости переформулировать без потери смысла.
- **Сокращения:** Использовать только общепринятые аббревиатуры (например, "ур." для "уровень", "HP" для "здоровье", если это привычно для игроков).
- **Избегать:** Не использовать длинные описания в интерфейсе, избегать повторов.

## 禁止模式 (Forbidden Patterns)

- **Запрещённые выражения:**
  - Сленг, жаргон, нецензурная лексика.
  - Избыточная формальность ("Уважаемый пользователь", "Настоящим уведомляем").
  - Кальки с китайского, неестественные для русского языка обороты.
  - Прямое копирование структуры китайских предложений.
- **Избегать:** Не использовать местоимение "ты" для обращения к игроку. Не допускать двусмысленности и неясности.

## 占位符处理 (Placeholder Handling)

- **Форматирование:** Сохранять все плейсхолдеры ({0}, {1}, <br> и т.д.) в исходном виде и на исходном месте в предложении.
- **Перенос строк:** <br> использовать только для разметки, не добавлять или удалять без необходимости.
- **Пояснения:** Если плейсхолдер обозначает переменную (например, имя, число), в переводе сохранять его позицию так, чтобы предложение звучало естественно на русском.
- **Пример:**  
  - Оригинал: "Вы получите {0} наград."  
  - Перевод: "Вы получите {0} наград."
- **Избегать:** Не изменять формат плейсхолдеров, не добавлять пробелы внутри фигурных скобок.

---

**Применяйте данный стиль для всех игровых текстов, чтобы обеспечить единообразие, удобство и максимальную понятность для русскоязычных игроков.**
